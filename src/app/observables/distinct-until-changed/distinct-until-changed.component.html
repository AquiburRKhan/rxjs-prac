Distinct Until Changed Operator:

Description:

It's like filter, but just emits the values that are distinct from the previous.

Examples 1:

of (1, 1, 2, 2, 2, 1, 1, 2, 3, 3, 4).pipe(
distinctUntilChanged(),
)
.subscribe() --->
1
2
1
2
3
4

Examples 2: DISTINCTUNTILCHANGED WITH OBJECTS

const sampleObject = { name: 'Test' };

const source$ = from([sampleObject, sampleObject, sampleObject]);

source$
.pipe(distinctUntilChanged())
.subscribe(x => console.log(x)); --->

{ name: 'Test' }

Examples 3: USING CUSTOM COMPARER FUNCTION

const source$ = from([
{ name: 'Brian' },
{ name: 'Joe' },
{ name: 'Joe' },
{ name: 'Sue' }
]);

.subscribe() --->

{ name: 'Brian' }
{ name: 'Joe' }
{ name: 'Sue' }
