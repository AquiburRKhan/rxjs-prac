Buffer Time Operator:

Description:

Collects values from the past as an array, and emits those arrays periodically in time.

Examples 1:

const source = interval(500);

const example = source.pipe(bufferTime(2000));

example.subscribe(val => console.log('Buffered with Time:', val) --->
Buffered with Time: [0]
Buffered with Time: [1 , 2]
Buffered with Time: [3 , 4]
Buffered with Time: [5 , 6]
