ConcatMap Operator:

Description:

Map values to inner observable, subscribe and emit in order / Projects each source value to an inner Observable which is merged in the output Observable, in a serialized fashion waiting for each one to complete before merging the next. concatMap does not subscribe to the next observable until the previous completes.

Examples 1:

const source = of(2000, 1000)

const example = source.pipe(
concatMap(val => of(`Delayed by: $ { val } ms`)
.pipe(delay(val)))
);
.subscribe(val => console.log(`With concatMap: ${ val }`))
With concatMap: Delayed by: 2000ms
With concatMap: Delayed by: 1000ms
