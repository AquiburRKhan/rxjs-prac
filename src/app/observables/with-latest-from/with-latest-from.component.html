WithLatestFrom Operator:

Description:

withLatestFrom combines each value from the source Observable (the instance) with the latest values from the other input Observables only when the source emits a value. All input Observables must emit at least one value before the output Observable will emit a value.

Code:

color$ = new Subject();
logo$ = new Subject();
color$.pipe(withLatestFrom(logo$))
.subscribe(([color, logo]) => console.log(`${color} shirt with ${logo}`));

Examples 1:

color$.next('white');
logo$.next('fish');
color$.next('green');
logo$.next('dog');
color$.next('red');
logo$.next('bird');
color$.next('blue'); --->

green shirt with fish
red shirt with dog
blue shirt with bird

Examples 2:

logo$.next('fish');
color$.next('white');
color$.next('green');
logo$.next('dog');
color$.next('red');
logo$.next('bird');
color$.next('blue'); --->

white shirt with fish
green shirt with fish
red shirt with dog
blue shirt with bird

Examples 3:

color$.next('white');
color$.next('green');
logo$.next('dog');
color$.next('red');
logo$.next('bird');
